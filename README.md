Task Manager App

A simple Task Manager web application built using HTML, CSS, and JavaScript. This project demonstrates basic frontend development concepts including DOM manipulation, event handling, and local storage.

Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Set priority levels (Low, Medium, High)
- Tasks persist using browser local storage
- Simple and responsive UI

Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

How It Works

- Tasks are stored in an array in JavaScript.
- Each task includes text, completion status, and priority.
- Data is saved in browser local storage so it persists after refresh.
- The UI updates dynamically using DOM manipulation.

Learning Outcomes

This project helped in understanding:

- JavaScript arrays and objects
- DOM manipulation
- Event handling
- Local storage usage
- Basic UI structuring and styling

Future Improvements

- Add due dates for tasks
- Add task editing feature
- Add drag and drop reordering
- Improve UI animations and transitions

This is a beginner-level project created for learning purposes.
